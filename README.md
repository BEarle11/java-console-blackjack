# Java Console Blackjack

A console-based implementation of the Blackjack card game written in Java.

## Features
- Standard Blackjack gameplay against a dealer
- Card and hand value calculation
- Player hit/stand decisions
- Dealer logic
- Win, loss, and tie detection

## Technologies Used
- Java
- Object-Oriented Programming
- Console input/output
- Game logic and state management

## How the Program Is Structured
- `Card` represents an individual playing card
- `Player` manages a hand of cards and score calculation
- `ConsoleBlackJackMain` controls the game loop and user interaction

## How to Run

### Run in an IDE (Recommended)
1. Open the project as a Java application
2. Run `ConsoleBlackJackMain.java`
3. Follow the console prompts

### Run from Terminal
Compile:
```bash
javac src/*.java
